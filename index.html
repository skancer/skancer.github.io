我对轮滑的认识

发表于 2019-03-31 | 分类于 轮滑 | | 阅读次数:
字数统计: 1.1k | 阅读时长 ≈ 3

轮滑是一项全身性运动的一项体育运动，轮滑比赛大致分为平地花式绕桩，刹车，速度过桩，battle（放大绝招）。即使带了护具也不能避免摔倒的伤害，所以滑行中一定要注意自己的安全。
当我们穿上轮滑鞋时，就可以体验惊险刺激的滑行感了，很多人喜欢轮滑，就是喜欢那种滑行感，自由，奔放的感觉。
1. 基础动作的练习
有句话说得好：上帝没有给我翅膀，我就用轮子自由飞翔。当你会滑之后，你便可以学一些轮滑的“技能”了，也就是一些基础的动作，就像英语中的背单词，是你今后学短语，句子，文章的基础。学动作的过程中是最艰辛的，也是我最愿意挑战的。学会了一个又一个动作就像打了一场又一场胜仗，但是不同的是，现在练的动作都是下一个动作的基础，并不是有一颗不服输的心就是可以把动作练成的，它需要有正确的方法指导，即使这样你也不一定能练成这个动作，因为可能是你上个动作仅仅是会了，而不熟练，这就需要你大量的练习了，成千上万的过桩，如果你还是练不成这个动作，你就要考虑你的身体基础了，因为它可能需要你的力量基础和你的韧带基础。如果你这些条件都具备了，还练不成这个动作的话，那就请你先换个动作练，过段时间再练。这就是练轮滑为什么会花那么多时间的原因。
2. 练习中的障碍
一方面怕苦怕累的人练不了，另一方面你还很容易进入练轮滑的误区，一旦进去，就需要一段时间来恢复，还有最严重的问题就是：坚持不下去，也是最常见的一个通病，原因大致有：

  1. 忽略基础的重要性，导致新动作很久都练不成，最后就成打酱油的了，最后甚至连酱油都不打了。
  2. 耐不住寂寞，真正的练习只需要你一个人，很多人在一起大部分时间都只能说是在玩，还有一部分干脆就直接谈起了恋爱，导致一方或双方的都消失。
  3. 学业对轮滑的影响。在我们学校，练轮滑的大三的基本上是很难见到的。那些四年一直坚持练下来的人真的很少，尤其在学校里，它需要一颗很强大的内心，很强大很强大的内心，像我就没有，我只是周期性的拥有。
    3. 轮子上的舞者
     等你的技能学的差不多了，你就可以练一些简单的组合动作，学别人的combo（就像英语作文中的范文），最后才能形成自己的风格。然后你就可以表现出来，轮滑它也是一项观赏性的运动。最终你都要编排自己的combo形成自己的style，里面有你想要表达的东西，要展现出来，展现出一种美，最好的境界就是你的动作当中要有自己的表达情感，甚至是灵魂，深深地感染观众。
    4. 最好的我们
    练了轮滑之后才发现很多的东西都是相通的，都是一个追求美的过程。最后说一下自己的感慨：现在的我们就是最好的。可能我们大都体会不到，感觉自己什么什么不好，很多事自己做的都不好，还有很多事是需要自己努力的等等，这些都是我们关心的。在这种种困难下，我们仍在面对着，努力去解决着，或许你感觉自己并没什么，但是，其实现在就是最好的自己。不管怎么样，在何种环境下，我们都需要做一棵永远成长的苹果树，这就是最好的我们。且行且珍惜。。。——2014.06.17

我的轮滑记录

发表于 2019-03-31 | 分类于 轮滑 | | 阅读次数:

从2012年进入大学选修轮滑课，第一次玩轮滑，2013年4月份加入俱乐部，到2015年12月到上海比完赛算是大学轮滑的结束吧！最后上一下我的战靴，现在还跟在我的身边，已经快四年了。

  1. 2013.04.27
    晚上时光何处花？ 喷泉广场滑轮滑。 汗水连连满地撒， 身体疲惫痛交加， 可这一切不算啥！ 只因由衷喜欢它。
  2. 2013.05.07
    有个 人陪你玩，固然是好，但自己一个人玩，也别有一番韵味，，，
  3. 2013.05.13
    我的“大头轮滑鞋”和它为我赢得的过杆冠军和障碍赛的亚军所得的奖品。纪念一下，或许以后再也不会穿上它了。
  4. 2013.06.13
    今天第一次穿轮滑鞋去刷街，去了阜阳的机场，路况很复杂，，，默默地滑了个来回。后来又到了老校区，坐在草地上聊天，一起在那说说笑笑的，不过聊起恐怖片的时候，我还是被吓到了。
  5. 2013.08.07
    “哥哥，你属什么的？”。“我属鸡的。”。“哇，属鸡的原来也可以这么厉害，我属猪的。”跟带着福建口音来的小朋友在一起轮滑，实在是太开心了
  6. 2013.08.14
    期待开学练轮滑， 生活琐事皆放下， 争取忘我只有它， 人轮合一方成家。
  7. 2013.08.20
    我也为家乡的轮滑普及做点贡献了。
  8. 2013.09.28
    如果你觉得自己过的很幸福，而你又将面临新的挑战，这将打破你以前的幸福，但你不需要患得患失（我在里面沉浸了好几天了，多亏了老师的指导，才走出来），那样只会让你什么也做不好，失去的更多。我们仅仅需要做的是，制定个合理的计划，融入新的生活，来迎接挑战，这样我们便能做的更好，过的更幸福…
  9. 2013.10.12
    虽然 还没有突破我 瓶颈期，但是在这一过程 ，我又学会了一些动作的新练法，人生不也是这样吗，（这也是我喜欢轮滑的一个原因：从中可以透露出很多哲理，不过都需要自己慢慢用心去体会感悟），这正如一道数学难题，其真正的意义并不是她的结果，而在于解决它的过程中却推动了数学的发展……
  10. 2013.10.16
    还是下点雨了，我也累了，练到现在的轮滑，单轮过了四次二十个桩的，一遍又一遍，一个又一个的，居然还把一个桩给踩坏了，那提神的啤酒真提神，太难喝了，现在我也可以找个地方休息下了……
  11. 2013.10.29
    之前对自己不喜欢的事，都很随意，慢慢的就对所有的事都随意了，包括自己喜欢的事情。那是一种很糟糕的生活状态，一切都只是为了应付生活。所以无论如何还都应该对生活认真点，首先做到对自己负责……
  12. 2013.11.07
    第一次早起晨练，一个人，两排桩。在桩与桩之间穿行，打破它们的孤寂

  1. 2013.11.25
    轮滑曾经是我的信仰，现在只想把它埋在心底。或许大多数玩家都是这样想的，我仅仅只是他们其中很普通的一个
  2. 2013.11.26
    练了有半年多的轮滑了，最近老是想着放弃，关键是因为最近一段时间的状态调整不好，以后每天晚上只练一个小时，不再东张西望，拖泥带水。再坚持一年，肯定会有让自己满意的结果等着我
  3. 2013.12.02
    心情不好，做了两千两百多个桩的crazy
  4. 2013.12.13
    2013年就要结束了，在这一年里，适应了大学生活，最重要的是选择了自己想要的生活了，即使累点苦点都不算什么，希望自己能够坚持的走下去！这一年里对我影响最大的还是轮滑，基本上排满了我的生活，我的性格和处事方式也收到了它很大的影响，同时也希望以后收获的更多，成为一名快乐的轮滑小将。最后说下学习，好多人觉得我是一个学霸，其实我并算不上，这里要自我批评下，我的学习并不踏实，我是主张乐学的。希望在以后的日子里能把学习态度端正，做一位称职的“学霸”。
  5. 2013.12.18
    当你知道自己应该做什么的时候应该就长大了。在这个过程中变的应该是周围的人和事，不变的是追求梦想的心。坚持自己喜欢的，我为自己加油！
  6. 2014.02.02
    自己都被对手感动了，我拿什么拯救自己！二级备战:轮滑。一级警告:自然熟的瓜，比乙烯催化的好
  7. 2014.02.21
    现在才发现，小时候自己是头驴，老师和父母牵着不走，打着后退。现在自己又是一头牛，被牵着走
  8. 2014.02.24
    以前经常想如果我没有练轮滑，那么省下的时间我可以干什么干什么，现在想一想人这一辈子没法做太多事情，所以我们应该把自己能做的，想做的，尽心尽力的做到精彩。另一方面体育锻炼可以磨练我们的意志，完善我们的人格，这点从书本知识上很难学到，真正喜欢体育的人可以赤裸裸的体会到。当我们为自己的生活创造出意义，便会得到自己精神世界上的真正满足。所以我愿意用这种方式度过自己绚丽多彩的一段人生。——献给曾经迷茫和将来迷茫时的自己。
  9. 2014.03.12
    夜深了，放下所有能放下的，去感受自己心脏的跳动，扑通扑通，一下两下，日日夜夜不辞辛苦的工作着，我就在想人生的真正意义究竟是什么呢？是期盼着充满未知的明天？还是沉浸在昨天的遗憾或快乐当中？应该是着眼于当下，努力走好现在的每一步。这样我们就应该会省下很多时间了，有些人经常说把今天当成自己的最后一天，我感觉这很容易让人误解，不如直接说，争取过好自己生活中的每一天，有时我们也总是高估了坚持一天的改变，低估了日复一日的积累对我们的改变，这就是一种平常心所拥有的平淡生活吧！明天又是新的一天，敬畏生命
  10. 2014.03.14
